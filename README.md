# bankapp
Банковское приложение
Простое консольное банковское приложение на Go.
Запустить через команду go run .
Что умеет программа:
 Создать счет (спросит имя)
 Пополнить счет
 Снять деньги
 Перевести на другой счет
 Посмотреть баланс
 Посмотреть историю операций
 Посмотреть все счета


Ошибки которые программа может найти:
 Пытаешься снять больше чем есть
 Вводишь отрицательную сумму
 Переводишь самому себе
 Счет не найден

 Все счета сохраняются в файл accounts.json в той же папке.
